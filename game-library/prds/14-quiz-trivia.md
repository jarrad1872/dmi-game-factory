## Core Mechanics

### Primary Action
- **Read question**: Player sees text question with 2-4 answer choices
- **Select answer**: Tap/click answer button within time limit
- **Instant feedback**: Correct answer highlighted immediately
- **Score points**: Points awarded based on speed and accuracy

### Question Types
1. **Multiple Choice** (4 options) - Primary format
2. **True/False** (2 options) - Quick fire rounds
3. **Image Identification** - "Which tool is this?"
4. **Spec Matching** - "Which spec matches this product?"
5. **Application Scenarios** - "Which tool for this job?"

### Win Conditions
- Complete quiz round (10-20 questions)
- Achieve minimum score threshold
- Beat opponent in head-to-head mode
- Unlock all categories

### Fail Conditions
- Score below passing threshold (in certification mode)
- Lose to opponent in competitive mode
- Run out of time (time attack mode)

### Progression System
- **Categories**: 6 main topic areas
  - Basic Products (Core bits, blades)
  - Specifications & Dimensions
  - Applications & Use Cases
  - Safety & Best Practices
  - DMI History & Brand
  - Advanced Technical
- **Difficulty Levels**: Easy → Medium → Hard → Expert
- **Unlocks**: Complete category to unlock next
- **Mastery**: 3-star rating per category
- **Overall Level**: Gain XP, level up, unlock titles

### Game Loop
1. Select category or random quiz
2. Question appears with 4 answer buttons
3. Timer starts counting down (10-30 seconds)
4. Player selects answer
5. Immediate feedback (correct/wrong + explanation)
6. Score updated, next question loads
7. After 10 questions: Summary screen with score, accuracy, speed
8. Option to replay, try new category, or challenge friend

```javascript
class QuestionManager {
  constructor(scene) {
    this.scene = scene;
    this.questionDatabase = null;
    this.currentCategory = null;
    this.currentDifficulty = 'medium';
    this.askedQuestions = [];
    this.questionPool = [];
    
    this.loadQuestions();
  }
  
  loadQuestions() {
    this.questionDatabase = this.scene.cache.json.get('questions_db');
    // Structure: { categories: { basic_products: [...], specs: [...], ... } }
  }
  
  startQuiz(category, questionCount = 10, difficulty = 'medium') {
    this.currentCategory = category;
    this.currentDifficulty = difficulty;
    this.askedQuestions = [];
    
    // Build pool of questions for this category/difficulty
    const categoryQuestions = this.questionDatabase.categories[category];
    this.questionPool = categoryQuestions
      .filter(q => q.difficulty === difficulty || difficulty === 'mixed')
      .slice();
    
    Phaser.Utils.Array.Shuffle(this.questionPool);
    this.questionPool = this.questionPool.slice(0, questionCount);
  }
  
  getNextQuestion() {
    if (this.questionPool.length === 0) return null;
    const question = this.questionPool.shift();
    this.askedQuestions.push(question);
    return question;
  }
  
  checkAnswer(questionId, answerIndex) {
    const question = this.askedQuestions.find(q => q.id === questionId);
    return answerIndex === question.correctAnswer;
  }
  
  getExplanation(questionId) {
    const question = this.askedQuestions.find(q => q.id === questionId);
    return question.explanation || 'No explanation available.';
  }
  
  getQuizResults() {
    let correct = 0;
    let totalTime = 0;
    
    this.askedQuestions.forEach(q => {
      if (q.userAnswer === q.correctAnswer) correct++;
      totalTime += q.timeSpent || 0;
    });
    
    const accuracy = (correct / this.askedQuestions.length) * 100;
    const avgTime = totalTime / this.askedQuestions.length;
    
    return {
      correct,
      total: this.askedQuestions.length,
      accuracy,
      avgTime,
      score: this.calculateScore()
    };
  }
  
  calculateScore() {
    let score = 0;
    this.askedQuestions.forEach(q => {
      if (q.userAnswer === q.correctAnswer) {
        // Base points: 100
        let points = 100;
        
        // Speed bonus (max 50 points)
        const timeBonus = Math.max(0, 50 - (q.timeSpent || 30));
        points += timeBonus;
        
        // Difficulty multiplier
        const difficultyMult = {easy: 1, medium: 1.5, hard: 2, expert: 3};
        points *= difficultyMult[q.difficulty] || 1;
        
        score += Math.floor(points);
      }
    });
    return score;
  }
}
```

```javascript
class QuizGameScene extends Phaser.Scene {
  create() {
    this.questionManager = new QuestionManager(this);
    this.scoreManager = new ScoreManager(this);
    this.timerBar = null;
    this.currentQuestion = null;
    this.questionIndex = 0;
    
    // Start quiz
    this.questionManager.startQuiz('basic_products', 10, 'medium');
    
    this.setupUI();
    this.showNextQuestion();
  }
  
  setupUI() {
    // Background
    this.add.image(400, 300, 'quiz_background');
    
    // Header
    this.add.rectangle(400, 40, 800, 80, 0x0047AB);
    this.categoryText = this.add.text(50, 40, 'CATEGORY: BASIC PRODUCTS', {
      fontSize: '18px',
      color: '#ffffff'
    });
    this.scoreText = this.add.text(650, 40, 'SCORE: 0', {
      fontSize: '24px',
      fontStyle: 'bold',
      color: '#F1C40F'
    });
    
    // Progress indicator
    this.progressText = this.add.text(400, 20, 'Question 1/10', {
      fontSize: '16px',
      color: '#ffffff'
    }).setOrigin(0.5);
    
    // Timer bar
    this.timerBarBg = this.add.rectangle(400, 90, 760, 20, 0x34495E);
    this.timerBar = this.add.rectangle(20, 90, 760, 20, 0x27AE60);
    this.timerBar.setOrigin(0, 0.5);
    
    // Question card
    this.questionCard = this.add.container(400, 200);
    const cardBg = this.add.rectangle(0, 0, 700, 200, 0xffffff);
    cardBg.setStrokeStyle(3, 0x95A5A6);
    
    this.questionText = this.add.text(0, 0, '', {
      fontSize: '24px',
      fontFamily: 'Roboto',
      color: '#2C3E50',
      align: 'center',
      wordWrap: {width: 650}
    }).setOrigin(0.5);
    
    this.questionCard.add([cardBg, this.questionText]);
    
    // Answer buttons
    this.answerButtons = [];
    const buttonColors = [0x3498DB, 0xE67E22, 0x27AE60, 0x9B59B6];
    const buttonY = [400, 470, 540, 610];
    
    for (let i = 0; i < 4; i++) {
      const button = this.createAnswerButton(400, buttonY[i], i, buttonColors[i]);
      this.answerButtons.push(button);
    }
  }
  
  createAnswerButton(x, y, index, color) {
    const button = this.add.container(x, y);
    
    const bg = this.add.rectangle(0, 0, 700, 60, color);
    bg.setInteractive();
    bg.setStrokeStyle(3, 0x34495E);
    
    const letter = ['A', 'B', 'C', 'D'][index];
    const letterBg = this.add.circle(-320, 0, 25, 0x2C3E50);
    const letterText = this.add.text(-320, 0, letter, {
      fontSize: '24px',
      fontStyle: 'bold',
      color: '#ffffff'
    }).setOrigin(0.5);
    
    const answerText = this.add.text(-280, 0, '', {
      fontSize: '20px',
      fontFamily: 'Montserrat',
      color: '#ffffff',
      align: 'left',
      wordWrap: {width: 600}
    }).setOrigin(0, 0.5);
    
    button.add([bg, letterBg, letterText, answerText]);
    
    // Hover effect
    bg.on('pointerover', () => {
      this.tweens.add({
        targets: bg,
        scaleY: 1.05,
        duration: 100
      });
    });
    
    bg.on('pointerout', () => {
      this.tweens.add({
        targets: bg,
        scaleY: 1,
        duration: 100
      });
    });
    
    bg.on('pointerdown', () => {
      this.onAnswerSelected(index, button);
    });
    
    return {container: button, bg, answerText, index};
  }
  
  showNextQuestion() {
    const question = this.questionManager.getNextQuestion();
    if (!question) {
      this.endQuiz();
      return;
    }
    
    this.currentQuestion = question;
    this.questionIndex++;
    this.questionStartTime = Date.now();
    
    // Update UI
    this.progressText.setText(`Question ${this.questionIndex}/10`);
    this.questionText.setText(question.question);
    
    // Show answers
    this.answerButtons.forEach((btn, i) => {
      btn.answerText.setText(question.answers[i]);
      btn.bg.setFillStyle(btn.bg.originalColor || [0x3498DB, 0xE67E22, 0x27AE60, 0x9B59B6][i]);
      btn.bg.setInteractive();
    });
    
    // Animate question in
    this.questionCard.setX(1000);
    this.tweens.add({
      targets: this.questionCard,
      x: 400,
      duration: 400,
      ease: 'Back.easeOut'
    });
    
    // Start timer
    this.startTimer(question.timeLimit || 20);
  }
  
  startTimer(duration) {
    const startWidth = 760;
    this.timerBar.width = startWidth;
    this.timerBar.setFillStyle(0x27AE60);
    
    this.timerTween = this.tweens.add({
      targets: this.timerBar,
      width: 0,
      duration: duration * 1000,
      ease: 'Linear',
      onUpdate: () => {
        const pct = this.timerBar.width / startWidth;
        if (pct < 0.3) {
          this.timerBar.setFillStyle(0xE74C3C); // Red warning
        } else if (pct < 0.5) {
          this.timerBar.setFillStyle(0xE67E22); // Orange
        }
      },
      onComplete: () => {
        // Time's up!
        this.onAnswerSelected(-1, null); // -1 = no answer (wrong)
      }
    });
  }
  
  onAnswerSelected(answerIndex, button) {
    // Stop timer
    if (this.timerTween) this.timerTween.stop();
    
    // Record time spent
    const timeSpent = (Date.now() - this.questionStartTime) / 1000;
    this.currentQuestion.timeSpent = timeSpent;
    this.currentQuestion.userAnswer = answerIndex;
    
    // Check if correct
    const correct = this.questionManager.checkAnswer(this.currentQuestion.id, answerIndex);
    
    // Disable all buttons
    this.answerButtons.forEach(btn => btn.bg.removeInteractive());
    
    // Show feedback
    const correctIndex = this.currentQuestion.correctAnswer;
    
    if (correct) {
      // Correct answer animation
      button.bg.setFillStyle(0x2ECC71);
      this.cameras.main.flash(200, 46, 204, 113, false);
      this.sound.play('correct_answer', {volume: 0.6});
      
      // Update score
      const points = this.calculatePoints(timeSpent, this.currentQuestion.difficulty);
      this.scoreManager.addScore(points);
      
      // Confetti effect
      this.playConfetti(button.container.x, button.container.y);
      
    } else {
      // Wrong answer animation
      if (button) {
        button.bg.setFillStyle(0xE74C3C);
        this.tweens.add({
          targets: button.container,
          x: button.container.x - 10,
          duration: 50,
          yoyo: true,
          repeat: 2
        });
      }
      
      // Show correct answer
      this.answerButtons[correctIndex].bg.setFillStyle(0x2ECC71);
      this.cameras.main.shake(100, 0.005);
      this.sound.play('wrong_answer', {volume: 0.5});
    }
    
    // Show explanation
    this.time.delayedCall(1500, () => {
      this.showExplanation(correct);
    });
  }
  
  showExplanation(wasCorrect) {
    const explanation = this.questionManager.getExplanation(this.currentQuestion.id);
    
    // Overlay
    const overlay = this.add.rectangle(400, 300, 800, 600, 0x000000, 0.7);
    const panel = this.add.container(400, 300);
    
    const bg = this.add.rectangle(0, 0, 600, 300, 0xffffff);
    bg.setStrokeStyle(4, wasCorrect ? 0x2ECC71 : 0xE74C3C);
    
    const title = this.add.text(0, -100, wasCorrect ? '✓ CORRECT!' : '✗ INCORRECT', {
      fontSize: '32px',
      fontStyle: 'bold',
      color: wasCorrect ? '#2ECC71' : '#E74C3C'
    }).setOrigin(0.5);
    
    const explanationText = this.add.text(0, -20, explanation, {
      fontSize: '18px',
      color: '#2C3E50',
      align: 'center',
      wordWrap: {width: 550}
    }).setOrigin(0.5);
    
    const continueBtn = this.createButton(0, 100, 'Continue', () => {
      panel.destroy();
      overlay.destroy();
      this.showNextQuestion();
    });
    
    panel.add([bg, title, explanationText, continueBtn]);
    
    panel.setScale(0);
    this.tweens.add({
      targets: panel,
      scale: 1,
      duration: 300,
      ease: 'Back.easeOut'
    });
  }
  
  calculatePoints(timeSpent, difficulty) {
    let points = 100;
    const timeBonus = Math.max(0, 50 - (timeSpent * 2));
    points += timeBonus;
    
    const diffMult = {easy: 1, medium: 1.5, hard: 2, expert: 3};
    points *= diffMult[difficulty] || 1;
    
    return Math.floor(points);
  }
  
  endQuiz() {
    const results = this.questionManager.getQuizResults();
    this.scene.start('QuizResultsScene', {results});
  }
  
  createButton(x, y, text, callback) {
    const button = this.add.container(x, y);
    const bg = this.add.rectangle(0, 0, 200, 50, 0x0047AB);
    bg.setInteractive();
    bg.on('pointerdown', callback);
    
    const label = this.add.text(0, 0, text, {
      fontSize: '20px',
      color: '#ffffff'
    }).setOrigin(0.5);
    
    button.add([bg, label]);
    return button;
  }
  
  playConfetti(x, y) {
    const emitter = this.add.particles(x, y, 'particle', {
      speed: {min: 200, max: 400},
      angle: {min: 0, max: 360},
      scale: {start: 1, end: 0},
      lifespan: 1000,
      quantity: 20
    });
    
    this.time.delayedCall(1000, () => emitter.destroy());
  }
}
```

```javascript
class ScoreManager {
  constructor(scene) {
    this.scene = scene;
    this.score = 0;
    this.xp = 0;
    this.level = 1;
    this.achievements = [];
  }
  
  addScore(points) {
    this.score += points;
    this.xp += points;
    this.scene.scoreText.setText(`SCORE: ${this.score}`);
    
    // Animate score increase
    this.scene.tweens.add({
      targets: this.scene.scoreText,
      scale: 1.2,
      duration: 150,
      yoyo: true

    this.checkLevelUp();
  }
  
  checkLevelUp() {
    const xpThresholds = [0, 100, 300, 600, 1000, 1500, 2500];
    const currentLevel = xpThresholds.findIndex((threshold, i) => 
      this.xp >= threshold && this.xp < (xpThresholds[i + 1] || Infinity)
    );
    
    if (currentLevel > this.level) {
      this.level = currentLevel;
      this.showLevelUp();
    }
  }
  
  showLevelUp() {
    // Level up animation and notification
    this.scene.sound.play('level_up');
    // ... show level up modal
  }
}
```

## PRD Summary

**Diamond Quiz: Expert Certification Challenge** transforms DMI's product knowledge base into an engaging trivia game that doubles as employee training and customer education. Players progress from "New Hire" to "Diamond Expert" by completing quiz challenges across six categories—from basic product recognition to advanced technical knowledge—earning certifications, badges, and XP along the way.

The game leverages proven quiz mechanics (multiple choice, timer pressure, instant feedback) while embedding real educational value. Each question teaches something about DMI's products, whether it's identifying the right tool for a job, understanding technical specifications, or mastering safety protocols. The immediate feedback system with explanations ensures players learn from both correct and incorrect answers, transforming mistakes into teaching moments.

Built on Phaser 3 with colorful UI, satisfying animations (confetti for correct answers, helpful explanations for wrong ones), and multiple game modes (Quick Quiz, Certification Tests, Daily Challenges, Time Attack, Head-to-Head), the game offers variety beyond repetitive drilling. The progression system—with XP, levels, titles, and badges—creates long-term engagement, while leaderboards add healthy competition among employees or customers. With 200+ carefully crafted questions across difficulty levels, Diamond Quiz provides weeks of content for onboarding new employees, engaging trade show visitors, or simply keeping the DMI brand top-of-mind. It's product training disguised as entertainment—and remarkably effective at both.
