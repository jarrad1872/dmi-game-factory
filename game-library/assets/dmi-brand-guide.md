## Typography

### Font Families

#### Body Text
- **Font:** Halant (serif)
- **Fallback:** serif
- **Usage:** Primary body copy, paragraphs, general text content
- **Style:** Classic, professional serif for readability

#### Display Headings (H1, H2)
- **Font:** Fraunces (serif)
- **Fallback:** serif
- **Usage:** Large headlines, hero text, major section headers
- **Style:** Bold, elegant serif with strong presence

#### Subheadings (H2, H3)
- **Font:** Open Sans (sans-serif)
- **Fallback:** sans-serif
- **Usage:** Section headers, navigation, UI elements
- **Style:** Clean, modern sans-serif for clarity

### Typography Hierarchy
```
H1: Fraunces (Display/Hero)
H2: Fraunces or Open Sans (Section Headers)
H3: Halant (Subheadings)
Body: Halant (Content)
UI/Nav: Open Sans (Interface)
```

## Design Guidelines for Game Integration

### Recommended Applications

#### Color Usage in Games
- Use **DMI Red (#A62022)** for:
  - Score highlights
  - Achievement badges
  - Power-ups
  - Primary buttons
  - Progress indicators

- Use **Rich Black (#222222)** for:
  - Text overlays
  - UI frames
  - Menu backgrounds (with opacity)

- Use **Light Gray (#EDEDED)** for:
  - Card backgrounds
  - Menu panels
  - Info boxes

#### Typography in Games
- **Headlines:** Fraunces Bold
- **UI Elements:** Open Sans
- **Body/Instructions:** Halant Regular

#### Thematic Elements
- Industrial/construction theme
- Concrete textures and patterns
- Diamond shapes and geometric patterns
- Tool silhouettes
- Professional jobsite aesthetics
- American flag elements (Made in USA)

#### Brand Integration
- Display DMI logo on splash screens
- Use "Made in the USA" as achievement theme
- Product categories as level names
- Construction site backgrounds
- Diamond blade/core bit inspired game mechanics
